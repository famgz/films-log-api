Flask API Documentation

This is the documentation for the Flask API. The API allows users to manage users, films, reviews, ratings, and favorites.

Endpoints

User Endpoints

POST /user

Create a new user.

Request Parameters (form-data):
  • username: string
  • email: string
  • password: string
Response: 
{"message": "User criado com sucesso"}
GET /user/<int:id>

Get a user by ID.

Response: 
{
    "id": int,
    "username": string,
    "email": string,
    "password": string
}
PUT /user/<int:id>

Update a user by ID.

Request Parameters (form-data):
  • username: string
  • email: string
  • password: string
Response: 
{"message": "User atualizado com sucesso"}
DELETE /user/<int:id>

Delete a user by ID.

Response: 
{"message": "User excluído com sucesso"}

Film Endpoints

POST /film

Create a new film.

Request Parameters (form-data):
  • title: string
  • year: int
  • director: string
  • duration: int

{"message": "Filme adicionado com sucesso"}

Response: 
[
