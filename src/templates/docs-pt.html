Documentação da API Flask

Esta é a documentação para a API Flask. A API permite que os usuários gerenciem usuários, filmes, resenhas, avaliações e favoritos.

Endpoints

Endpoints de Usuário

POST /user

Cria um novo usuário.

Parâmetros da Requisição (form-data):
  • username: string
  • email: string
  • password: string
Resposta: 
{"message": "User criado com sucesso"}
GET /user/<int:id>

Obter um usuário pelo ID.

Resposta: 
{
    "id": int,
    "username": string,
    "email": string,
    "password": string
}
PUT /user/<int:id>

Atualizar um usuário pelo ID.

Parâmetros da Requisição (form-data):
  • username: string
  • email: string
  • password: string
Resposta: 
{"message": "User atualizado com sucesso"}
DELETE /user/<int:id>

Excluir um usuário pelo ID.

Resposta: 
{"message": "User excluído com sucesso"}

Endpoints de Filme

POST /film

Cria um novo filme.

Parâmetros da Requisição (form-data):
  • title: string
  • year: int
  • director: string
  • duration: int
Resposta: 
{"message": "Filme adicionado com sucesso"}
GET /film/all

Obter todos os filmes.

Resposta: 
[
    {
        "id": int,
        "title": string,
        "year": int,
        "director": string,
        "duration": int
    },
    ...
]
GET /film/<int:id>

Obter um filme pelo ID.

Resposta: 
{
    "id": int,
    "title": string,
    "year": int,
    "director": string,
    "duration": int
}
PUT /film/<int:id>

Atualizar um filme pelo ID.

Parâmetros da Requisição (form-data):
  • title: string
  • year: int
  • director: string
  • duration: int
Resposta: 
{"message": "Filme atualizado com sucesso"}
DELETE /film/<int:id>

Excluir um filme pelo ID.

Resposta: 
{"message": "Filme excluído com sucesso"}

{"message": "Favorite adicionado com sucesso"}
