Documentação do Films Log API

Usuários

POST /user

Cria um novo usuário.

GET /user/{username}

Obtém informações de um usuário.

PUT /user/{username}

Atualiza informações de um usuário.

DELETE /user/{username}

Exclui um usuário.

GET /user/{username}/film/all

Obtém todos os filmes de um usuário.

GET /user/{username}/film/{film_id}

Obtém informações de um filme específico de um usuário.

Filmes

POST /film

Adiciona um novo filme.

GET /film/all

Obtém todos os filmes.

GET /film/{id}

Obtém informações de um filme por ID.

PUT /film/{id}

Atualiza informações de um filme.

DELETE /film/{id}

Exclui um filme.

Reviews

POST /user/{username}/film/{film_id}/review

Cria uma nova review para um filme.

GET /user/{username}/film/{film_id}/review

Obtém a review de um filme feita por um usuário.

PUT /user/{username}/film/{film_id}/review

Atualiza a review de um filme feita por um usuário.

DELETE /user/{username}/film/{film_id}/review

Exclui a review de um filme feita por um usuário.

Ratings

POST /user/{username}/film/{film_id}/rating

Cria um novo rating para um filme.

GET /user/{username}/film/{film_id}/rating

Obtém o rating de um filme feito por um usuário.

PUT /user/{username}/film/{film_id}/rating

Atualiza o rating de um filme feito por um usuário.

DELETE /user/{username}/film/{film_id}/rating

Exclui o rating de um filme feito por um usuário.

Favoritos

POST /user/{username}/film/{film_id}/favorite

Marca um filme como favorito para um usuário.

GET /user/{username}/film/{film_id}/favorite

Obtém se um filme é favorito para um usuário.

PUT /user/{username}/film/{film_id}/favorite

Atualiza se um filme é favorito para um usuário.

DELETE /user/{username}/film/{film_id}/favorite

Exclui a marcação de um filme como favorito para um usuário.
